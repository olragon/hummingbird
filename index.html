Hummingbird next (WIP)

Hummingbird lets you see how visitors are interacting with your website in real time.

And by “real time” we donâ€™t mean it refreshes every 5 minutes—WebSockets enable Hummingbird to update 20 times per second.

Hummingbird is built on top of Node.js, a set of event-driven javascript web libraries built on top of the V8 javascript engine.

Hummingbird-next is attempted to made Hummingbird project great again.

  • using with express or node's raw server
  • upgrade required modules

How It works

Downloading & Installation

npm install hummingbird-next --save

How to use

Copy the default configuration file: 

var http = require('http');
var express = require('express');
var app = express();
var server = http.Server(app);
var hummingbird = require('hummingbird-next');

// hummingbird dashboard will be ready at /dashboard/
hummingbird.dashboard.attach(server, '/dashboard/');

// tracking pixel will be ready at /tracking_pixel.gif
hummingbird.tracker.attach(server, '/tracking_pixel.gif'); 

var listener = server.listen(process.env.PORT || 3000, function () {
  console.log('Your app is listening on port ' + listener.address().port);
});

Deployment

Hummingbird will deploy pretty much anywhere that supports node.js and websockets.
